Tentang Saya

👤
foto-saya.jpg

Halo! Saya adalah Junior Data Analyst yang bersemangat dalam mengeksplorasi data dan menemukan pola tersembunyi yang bisa menggerakkan keputusan bisnis.

Latar belakang saya di Data Analyst membentuk cara saya berpikir analitis — setiap dataset adalah sebuah cerita yang menunggu untuk diceritakan. Saya percaya bahwa data yang baik, dikomunikasikan dengan baik, bisa mengubah arah perusahaan.

Ketika tidak sedang menggali data, saya aktif mengikuti komunitas data lokal dan terus belajar teknik-teknik baru di bidang machine learning dan statistik.

Python (Pandas, NumPy) SQL (PostgreSQL, MySQL) Tableau / Power BI Google Sheets / Excel Matplotlib / Seaborn Git & GitHub
