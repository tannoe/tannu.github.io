Pharmaceutical Sales Performance Analysis
Role: Individual project · RevoU · Apr–Jun 2026
Background
Dataset 254.000+ transaksi & retur produk B2B senilai agregat $11.95B dari 372 klien aktif, kondisi data masih mentah dan belum terstandarisasi.
Method
Membersihkan dan menggabungkan data menggunakan Python (Pandas), lalu membangun dashboard Tableau untuk memvisualisasikan performa penjualan multi-tahun di 2 region.
Result
Dashboard mengidentifikasi tren penjualan antar region yang mendukung pengambilan keputusan strategis manajemen sales.

Revo Hospital Dashboard
Role: Individual project · RevoU · May 2026
Background
5.000 data operasional rumah sakit (2024–2025) dari beberapa cabang, dibutuhkan evaluasi performa layanan kesehatan antar cabang.
Method
Menganalisis tingkat okupansi tiap cabang menggunakan Tableau dan Spreadsheet.
Result
Menemukan cabang dengan okupansi hingga ~160% (indikasi over-capacity), lalu mengusulkan 3 rekomendasi strategis (optimasi kapasitas, manajemen discharge, alokasi sumber daya) dengan potensi efisiensi operasional 10%.
